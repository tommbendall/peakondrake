===========
Peakondrake
===========

Peakondrake is a small package using the Firedrake software that simulates
the stochastic Camassa-Holm equation, in various formulations, using finite
element discretisations.

============
INSTALLATION
============

Must be run inside the Firedrake virtualenv. The path to peakondrake can be
added by running:

python3 setup.py install
